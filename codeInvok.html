2. 接入代码

2.1.1 配置appkey和广告位id

在添加string.xml文件添加appkey与广告位id


     <resources>
     <string name="app_name">adsdkDemo</string>

     <string name="app_key">670cb5671c9949a0bbc69d5404604965</string>
     <string name="banner_ad_unitid">5a756e33373142c99dcee61f70006873</string>
     <string name="intersitial_ad_unitid">ba3f58d2dc03456b8b6e5dc97f4f1241</string>
     <string name="native_ad_unitid">19386f9531c941cbbccd0b49ee193106</string>
     </resources>

※    appkey与广告位id由运营提供。以上实例广告位为测试广告位id仅用于测试时使用

2.1.2 调用初始化方法

在app的入口Application里调用sdk初始化代码(appkey和广告位ID从广告管理平台申请)

        /**
        * Initialize the Mobile Ads SDK. The SDK will configure itself.
        * @param context the context running in
        * @param applicationCode appkey
        * @param unitIds mutil unit Id string
        */
        HkMobileAds.initialize(getApplicationContext(),getString(R.string.app_key),getString(R.string.banner_ad_unitid),getString(R.string.intersitial_ad_unitid));
